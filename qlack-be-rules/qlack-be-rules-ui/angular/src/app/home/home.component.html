Welcome to the Rules component

The Rules module allows QBE users to create and manage the business rules required by their application. More specifically, QBE users are able to perform the following actions:

  • Create business rules, either through the provided graphical interface (simple rules) or by uploading a DMN (.xml) file which can later be edited through the provided text editor (complex rules).
  • Edit business rules including creating and managing different versions of each business rule.
  • Specify the data models used during business rule execution, either through the UI or by uploading Java classes.
  • Group different versions of business rules and data models in working sets, categories and tags for easier reference.

Business rules can be used to perform validation of a form created in the Forms component or as part of the execution of a workflow created in the Workflow component.
